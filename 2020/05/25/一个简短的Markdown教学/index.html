什么是Markdown，我为什么要学习Markdown

Markdown是一种轻量级标记语言，多用在个人博客(例如：本博客、CSDN)、小说站、笔记软件(例如：印象笔记、有道云笔记)
它们通常为.md或.markdown的后缀名称，并且不止一种主题，例如GitHub上就有多种主题可选择
它可以非常快速的记录一篇笔记或者写出一篇文章并导出为Epub Word PDF HTML甚至是图片，你不需要任何的语言基础都可以使用一些基本功能
